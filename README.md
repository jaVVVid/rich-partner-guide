# rich-partner-guide

Part of the **Rich Skills** series for OpenClaw.

A public OpenClaw skill for running an assistant in a practical partner style: concise communication, clean execution boundaries, memory hygiene, phased work, and shared-chat restraint.

## Series Notes

- Author: Rich × Javid
- Collection: Rich Skills for OpenClaw
- Related repositories:

  - [`rich-self-improvement`](https://github.com/jaVVVid/rich-self-improvement)

## What it does

- Pushes the assistant toward answer-first replies
- Encourages concise, high-signal communication
- Promotes semantic memory retrieval before acting on prior context
- Defines phased execution and milestone-style updates
- Adds safety boundaries for risky, destructive, or external actions
- Helps assistants behave better in shared chats and heartbeat flows

## Best fit

Use this skill when you want an OpenClaw assistant to feel less like a generic chatbot and more like a careful, proactive partner.

## Contents

- `SKILL.md` — main skill definition
- `references/` — supporting playbooks and patterns
- `scripts/` — helper scripts for summaries and handoff rendering

## Install locally

Copy this folder into your OpenClaw workspace `skills/` directory, then enable or use it according to your setup.
